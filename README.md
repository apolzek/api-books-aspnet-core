# api-books-aspnet-core
Web API with ASP.NET  Core and MongoDB :heavy_check_mark: .NET Core SDK 5

## Prerequisites

- .NET Core SDK 5.0.103 or later
- Visual Studio Code 
- MongoDB(docker or service)
